# 🔌 繁星抓取 Chrome 扩展 · 安装指南

## 第一步:装扩展(1 分钟,装一次永久用)

1. 打开 Chrome,地址栏输入:`chrome://extensions/` 回车
2. 右上角打开「**开发者模式**」开关
3. 左上角点「**加载已解压的扩展程序**」
4. 选择这个文件夹:
   ```
   C:\Users\admin\Desktop\AI--\english-dashboard\chrome-extension
   ```
5. 装完浏览器右上角会多出一个「**繁**」字图标
6. 如果没看到,点拼图 🧩 图钉一下,固定到工具栏

## 第二步:用

1. Chrome 里登录小鹅通繁星,打开任意一节课
2. 点右上角「**繁**」扩展图标
3. 弹出窗口里点「🔍 **抓取当前页面**」
4. 扩展会扫描页面,列出所有识别到的中英对照
5. 点「🚀 **一键导入看板**」→ 自动:
   - 复制内容到剪贴板(兜底)
   - 打开英语看板
   - 自动填入「📥 今日输入」文本框
   - **自动保存**,立刻可以开练

## 第三步(可选):改看板地址

默认看板地址是本地文件:
```
file:///C:/Users/admin/Desktop/AI--/english-dashboard/index.html
```

如果你用 Python HTTP 服务器跑,改成:`http://localhost:8765/`

如果你部署到手机(PWA),填网址即可。改完下次用会记住。

---

## 🐛 如果抓不到内容怎么办

1. 先点扩展里的「🔧 **调试:看所有候选文本**」展开,看看页面文字到底被识别到没
2. 如果看到了文字但没配对 → 点「📋 **复制所有页面文字**」,粘给 Claude 说:  
   *"这是繁星某一页的原始文本,帮我把中英对照整理成看板能用的格式"*
3. 如果连文字都没抓到 → 可能页面是 Shadow DOM 或 iframe,把 URL 发给 Claude 适配

## ⚙️ 改进抓取规则

扩展的抓取逻辑在 `popup.js` 里的 `scrapePage()` 函数:
- 策略 A:一行里同时有 CN 和 EN(`确认 | confirm` 或 `确认 confirm`)
- 策略 B:兄弟节点分别是 CN 和 EN(DOM 中 `<p>确认</p><p>confirm</p>`)
- 策略 C:用 class 名暗示双语(`class="translate"`、`class="bilingual"` 等)

如果繁星用了特殊的 class,告诉 Claude 加一条专门规则就行。

---

## 🔒 隐私说明

- 扩展**只在你点图标时**运行,不在后台监听
- 抓取的内容**只存在 Chrome 扩展本地**和你的看板 localStorage
- 不联网,不上报,不发任何数据出去
- 权限只要了 `activeTab`(当前标签)和 `clipboardWrite`(复制到剪贴板)
